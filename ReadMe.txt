Программа "HelpProgramm" имитирует действие тех поддержки уже купленной продукции. Пользователь у которого имеется продукция, имеет свое приложение для отправки запроса в тех поддержку, о проблеме с купленной продукцией. Сотрудник тех поддержки, который имеет то же свое приложение может рассмотреть данную заявку и ответить пользователю. Программа была написана на следующих технологиях (WPF - для создания основных приложений,WCF-сервис для связи клиента и пользователя удаленно, MsSql для хранения информации).

Для того что бы программа работала требуется импортировать Бд в MsSql , она находится в папке New DB

После чего запустить сервис который находится в папке SupportService.beta_1
->
Потом уже запускать любую клиентскую часть
